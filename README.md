# Power BI Power Query ETL Project

## Overview

This project demonstrates an ETL (Extract, Transform, Load) process using Power BI and Power Query. Data was extracted from web tables, Excel files, and folders, then cleaned, transformed, merged, and analyzed for reporting purposes.

## Data Sources

* HTML table from web
* Monthly sales Excel files (Jan, Feb, Mar)
* Employee dataset Excel file
<img width="297" height="372" alt="image" src="https://github.com/user-attachments/assets/a99d0a3e-4ac7-4155-90da-6661edc8410a" />

## Transformations Applied

* Removed blanks and duplicates
* Promoted headers
* Renamed columns
* Changed data types
* Text cleaning using UPPER, LOWER, TRIM, CLEAN
* Created Profit column
* Added conditional columns and indexes
* Pivoted and unpivoted data
* Merged and appended tables
* Grouped data for aggregations

## Features

* Dynamic folder path using Parameters
* Data profiling and quality checks
* Auto-refresh simulation with new monthly file

## Key Learnings

* Power Query ETL workflow
* Data cleaning and preprocessing
* Merge & Append operations
* Aggregation and profiling
* Scalable refresh process

## Deliverables

* Power BI (.pbix) file
* README/Documentation file

## Conclusion

The project successfully demonstrates real-world Power BI ETL and data transformation techniques for building scalable analytical datasets.
